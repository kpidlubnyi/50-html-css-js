# 50 Mini Projects — HTML, CSS, JavaScript

Hi. This repository is a place where I store the mini projects I build while learning **HTML, CSS, and JavaScript**.

The goal is simple: learn front-end development by building many small projects.  
Each project focuses on a specific concept such as layout, styling, animations, or basic JavaScript interactions.

Inspiration comes from this repository:
https://github.com/bradtraversy/50projects50days

However, the projects here are written by me as part of my learning process.

## What you will find here

- Small front-end projects
- Experiments with HTML, CSS, and JavaScript
- Practice of DOM manipulation
- Simple UI components and interactions

## Purpose of this repository

- Practice the fundamentals of web development  
- Build a habit of learning by creating projects  
- Track my progress while studying front-end technologies  

## Structure

Each project usually contains:
```
project-name/
  index.html
  style.css
  script.js
```

Some projects may also include images or other assets.

This repository is mainly for **learning and practice**.
